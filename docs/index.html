How it works

⚖️
Smart Ranking

Answer a few plain-English questions about each task. Priority calculates which one to do first based on stakes, urgency, and effort.

🌊
Domino Effect

A small task that unblocks three big ones ranks higher than any of them alone. Downstream impact is factored in automatically.

🗺️
Critical Path

The longest chain of dependent tasks — any delay here delays everything downstream. These get a ★ marker after you run Analyze.
